解約忘れを防ぐ方法

VODの解約は公式サイトのアカウント設定から24時間いつでも可能です。電話手続きは不要で、所要時間は数分。違約金もかかりません。

  1. 公式サイトにログイン
  2. アカウント設定 → 契約情報 → 解約
  3. アンケートに回答（任意）
  4. 解約完了メールを確認

日割り返金がない点には注意。月末ギリギリの解約だと損をするため、月初に解約日を決めておくのがお得です。

❓ よくある質問

全VODの無料体験を順番に試してもいい？
問題ありません。各社で別アカウントを作成すれば期間中の解約で料金は発生しません。月をまたいで4社試せば4ヶ月分タダで楽しめます。
31日と30日で体感はどれくらい違う？
わずか1日の差ですが、ドラマシリーズを完走するには貴重。31日VODから順に試すのがおすすめです。
1日でも過ぎたら全額請求される？
はい、自動的に翌月分が引き落とされます。「期日午前中までに解約」を目安にしてください。
